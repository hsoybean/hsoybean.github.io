CentOS 7.x使用yum安装JDK1.8

Feb 7, 2023
1 min read
centOS Java 开发 技术

操作系统: Centos 7.x

一. 安装JDK

1. yum源查找JDK版本安装包

查找所有java关键字匹配的安装包

yum search java*

查找java1.8版本相关的安装包

yum search java-1.8

使用OpenJDK Development Environment对应的安装包：开发版本 JDK 不要使用OpenJDK Runtime Environment对应的安装包：只有java运行环境 JRE

2. 安装JDK

yum -y install java-1.8.0-openjdk-devel.x86_64

二. 配置环境变量

vim ~/.bash_profile

配置以下内容

export JAVA_HOME=/usr/lib/jvm/java-1.8.0-openjdk-1.8.0.282.b08-1.el7_9.x86_64
export CLASSPATH=.:$JAVA_HOME/lib
PATH=$JAVA_HOME/bin:$PATH

source一下，使配置生效

source ~/.bash_profile

测试是否安装完成： 任意目录下执行命令：javac java

安毕。。。。
