开源博客系统Halo安装及使用

Feb 7, 2023
1 min read
建站 博客 技术

操作系统: Centos 7.x

一. 安装宝塔面板

1. 更新yum

yum update

2. 使用yum安装面板

官方安装文档：https://www.bt.cn/bbs/thread-19376-1-1.html

yum install -y wget && wget -O install.sh 
http://download.bt.cn/install/install_6.0.sh && sh install.sh

出现以上信息，说明宝塔面板已经安装完成。 根据信息内容中的外网面板地址在浏览器打开,输入username和password登陆面板。（如果忘记username和password，可以在服务器linux命令行使用 bt default来查看） 第一次登陆进面板之后，会有个提示安装LNMP和LAMP的环境套件，这里我选择LNMP（linux Nginx mySql PHP），安装方式选择“极速安装”，点击一键安装。

二. 安装Docker

官网安装文档：https://docs.docker.com/engine/install/centos/#uninstall-old-versions

1. 卸载老版本的Docer

sudo yum remove docker \
	docker-client \
	docker-client-latest \
	docker-common \
	docker-latest \
	docker-latest-logrotate \
	docker-logrotate \
	docker-engine

2. 配置docker仓库

yum install -y yum-utils

sudo yum-config-manager \
		--add-repo \
    	https://download.docker.com/linux/centos/docker-ce.repo

3. 安装Docker引擎

sudo yum install docker-ce docker-ce-cli containerd.io

命令行显示“完毕”，此时Docker已经安装完成，但是还没有Docker服务还没有启动。

4. 启动Docker

sudo systemctl start docker

5. 运行一个hello-world镜像验证Docker是否正常运行

sudo docker run hello-world

6. 为Docker设置开机启动

systemctl enable docker

三. 安装Halo

官网安装文档：https://docs.halo.run/zh/install/index

以下选择用Docer的方式进行安装Halo：

1. 创建工作目录

mkdir ~/.halo && cd ~/.halo

2. 下载示例配置文件到工作目录(下载完成之后可以使用vimvim application.yaml 编辑配置文件，配置数据库或者端口等)

wget https://dl.halo.run/config/application-template.yaml -O ./application.yaml

3. 拉取最新的 Halo 镜像

docker pull halohub/halo

4. 创建容器

docker run -it -d --name halo -p 8090:8090 -v ~/.halo:/root/.halo --restart=always halohub/halo

5. Halo安装引导界面

打开http://外网ip:端口号，即可看到安装引导界面 如果打不开，请前往服务器的安全组开放8090端口后重试
